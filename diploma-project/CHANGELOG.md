#### Технические характеристики
- Точность модели: > 95%
- Время обучения: < 5 секунд
- Время предсказания: < 10ms
- Размер ML Docker образа: ~500 MB
- Размер Web Docker образа: ~20 MB
- Поддерживаемые платформы: Linux, macOS, Windows

#### API Endpoints

**ML Service:**
- `POST /predict` - предсказание специальностей
- `GET /health` - проверка здоровья
- `GET /specialties` - список специальностей

**Web Service:**
- `GET /` - главная страница
- `POST /api/predict` - получение предсказаний
- `POST /api/submit` - сохранение результатов
- `GET /api/results/:id` - просмотр результатов
- `GET /health` - проверка здоровья

## Известные ограничения

### Версия 1.0.0
- Модель обучена на синтетических данных
- Небольшой размер датасета (540 записей)
- Отсутствие аутентификации пользователей
- Нет rate limiting для API
- SQLite база (не подходит для высоких нагрузок)

### Рекомендации для production
1. Заменить SQLite на PostgreSQL/MySQL
2. Добавить Redis для кэширования
3. Настроить HTTPS с SSL сертификатами
4. Реализовать rate limiting
5. Добавить логирование и мониторинг (Prometheus + Grafana)
6. Внедрить аутентификацию (JWT)
